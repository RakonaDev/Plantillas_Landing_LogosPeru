# Plantillas Landing LogosPeru

Repositorio que agrupa 6 plantillas de landing page desarrolladas para distintos rubros comerciales. Cada plantilla es un proyecto independiente construido con React, TypeScript y Vite.

## Tecnologias utilizadas

Tecnologias base presentes en el repositorio:

- React 19
- TypeScript
- Vite
- Tailwind CSS
- PostCSS y Autoprefixer
- ESLint

Librerias usadas segun la plantilla:

- React Router DOM
- Motion para animaciones
- React Icons
- Lucide React
- Formik y Yup para formularios y validaciones
- Swiper para sliders y carruseles
- GSAP en animaciones puntuales

## Tipos de plantillas

| Plantilla | Tipo | Descripcion |
| --- | --- | --- |
| plantilla1 | Transporte y alquiler vehicular | Landing enfocada en alquiler de autos, vans, buses y servicios de movilidad. |
| plantilla2 | Ferreteria y catalogo comercial | Landing para ferreteria con secciones de productos, servicios y contacto. |
| plantilla3 | Arquitectura y diseno | Landing corporativa para estudio de arquitectura, diseno interior y proyectos. |
| plantilla4 | Turismo y agencia de viajes | Landing promocional para paquetes turisticos, vuelos, hoteles y excursiones. |
| plantilla5 | Salud y centro medico | Landing para clinica o centro medico con servicios especializados y contacto. |
| plantilla6 | Inmobiliaria y bienes raices | Landing para compra, venta y alquiler de propiedades. |
